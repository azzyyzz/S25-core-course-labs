# Moscow Time Web App

## Overview

Simple Go web app to show time in Moscow.

![image](static/images/overview.png)

## Installation

```bash
git clone https://github.com/azzyyzz/S25-core-course-labs.git
cd S25-core-course-labs/app_go
```

## Requirements

- [Go](https://golang.org/dl/) 1.16+

## Usage

```bash
go run app.go
```
